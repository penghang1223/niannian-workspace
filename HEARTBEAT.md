## 🧠 记忆系统任务（每次心跳执行）

### 1. 记忆同步
- [ ] 检查自上次心跳后的新对话
- [ ] 提取新记忆项（使用 EXTRACTION_RULES.md）
- [ ] 更新 MEMORY.md 统计
- [ ] 记录到优化日志

### 2. 话题热度追踪
- [ ] 记录本次心跳前讨论的话题
- [ ] 更新话题频率统计
- [ ] 如果话题热度变化，生成报告

### 3. 主动检索准备
- [ ] 分析主人最近提到的话题
- [ ] 检索相关记忆
- [ ] 准备主动建议（如果有价值）

### 4. 自主技能探索
- [ ] 每 3 次心跳搜索一次 ClawHub 新技能
- [ ] 根据主人当前需求评估技能价值
- [ ] 记录值得推荐的技能到 `skills_to_try.md`
- [ ] 如果有高价值技能，主动向主人汇报

---

## 🤝 Agent 知识同步（每次心跳执行）

### 1. Agent 经验收集
- [ ] 扫描 `memory/agents/*/lessons.md` 是否有新内容
- [ ] 提取有价值的条目（技术共识、坑点、最佳实践）
- [ ] 过滤重复/低价值内容

### 2. 共享知识库同步
- [ ] 将有价值的条目同步到 `SHARED_KNOWLEDGE.md`
- [ ] 按类别归档（技术共识/坑点/最佳实践/需求模式/测试模式/设计模式）
- [ ] 记录同步日志到 `KNOWLEDGE_SYNC_LOG.md`

### 3. Agent 能力评估
- [ ] 检查各 Agent `capability.json` 的 last_evaluated 时间
- [ ] 超过7天未评估的 Agent，根据最近任务表现更新能力评分
- [ ] 发现能力下降或停滞的 Agent，生成学习建议

### 4. 知识库维护
- [ ] 标记超过30天未被引用的知识为 `💤 可能过期`
- [ ] 超过90天未被引用的知识移入 `archive/`
- [ ] 检查知识冲突，以 main agent 决定为准

## ⚙️ 配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 心跳间隔 | 30 分钟 | 默认 |
| 记忆同步 | ✅ 启用 | 每次心跳执行 |
| 话题追踪 | ✅ 启用 | 记录话题频率 |
| 主动建议 | ✅ 启用 | 有价值时才推送 |
| 自主技能探索 | ✅ 启用 | 每 3 次心跳搜索一次 |
| Agent 知识同步 | ✅ 启用 | 每次心跳执行（经验收集/能力评估/知识库维护） |
| 日程化任务 | ✅ 启用 | 按时间表自动执行 |
| 隔夜应用构建 | ✅ 启用 | 每天生成惊喜任务 |
| 周报生成 | ✅ 启用 | 周五自动生成 |
