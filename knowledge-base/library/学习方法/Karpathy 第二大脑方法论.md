## 🎯 核心理念

### 从"用 AI 写代码"到"用 AI 建大脑"

**传统用法**：
```
用户提问 → AI 回答 → 对话结束 → 知识丢失 ❌
```

**Karpathy 用法**：
```
用户丢素材 → AI 整理成 Wiki → 持续积累 → 随时查询 ✅
```

---

## 🏗️ 三层架构

```
llm-wiki/
├── raw/          # 原始素材（不可变，只增不减）
├── library/      # AI 整理的知识库（维基百科）
└── output/       # 产出内容（报告/代码/文章）
```

---

## 📁 raw/ - 原始素材层

### 原则
- ✅ 只增不减
- ✅ 保持原始状态
- ✅ 不手动编辑

### 存放内容
| 类型 | 示例 |
|------|------|
| 论文 | PDF/arXiv 文章 |
| 代码 | GitHub 仓库/代码片段 |
| 文章 | 技术博客/新闻 |
| 数据集 | CSV/JSON数据 |
| 会议记录 | 转录文本 |
| 对话 | 重要聊天记录 |

### 命名规范
```
raw/<日期>_<主题>_<描述>.md
示例：
- 2026-04-08_Karpathy_LLM Wiki 架构.md
- 2026-04-08_OpenClaw_多 Agent 协作.md
```

---

## 📚 library/ - 知识库层

### 核心概念
**AI 作为"全职图书管理员"**：
- 阅读 raw/ 素材
- 提取知识点
- 创建 Markdown 文章
- 建立反向链接
- 持续更新维护

### 文章结构
```markdown
# 文章标题

## 📎 来源
- [[raw/2026-04-08_xxx.md]]

## 📝 核心内容
...

## 🔗 相关条目
- [[相关文章 1]]
- [[相关文章 2]]

## 🔄 更新历史
- 2026-04-08: 初始版本
```

### 关键特性
| 特性 | 说明 | 价值 |
|------|------|------|
| **人类可读** | Markdown 格式 | 直接打开编辑 |
| **自我修复** | AI 自动更新 | 无需手动维护 |
| **可审计** | 每篇有来源 | 追溯原始素材 |
| **反向链接** | 知识自动关联 | 发现隐藏联系 |
| **无需向量库** | 简化架构 | 降低成本 |

## 📤 output/ - 产出层

### 存放内容
| 类型 | 示例 |
|------|------|
| 报告 | 学习报告/分析报告 |
| 代码 | 项目代码/脚本 |
| 文章 | 博客文章/教程 |
| 总结 | 读书笔记/会议总结 |

### 创作流程
```
基于 library/知识库
    ↓
AI 辅助创作
    ↓
output/产出内容
    ↓
读者反馈 → raw/ → 循环优化
```

---

## 🔄 工作流程

### 1. 收集素材
```
看到好文章/论文/代码
    ↓
丢入 raw/文件夹
```

### 2. AI 整理
```
LLM 阅读 raw/素材
    ↓
提取知识点
    ↓
创建/更新 library/文章
    ↓
建立反向链接
```

### 3. 创作产出
```
基于 library/知识库
    ↓
AI 辅助创作
    ↓
output/产出内容
```

### 4. 反馈循环
```
读者反馈/新学习
    ↓
添加到 raw/
    ↓
更新 library/
    ↓
优化 output/
```

## 🆚 vs 传统 RAG

| 对比项 | RAG | LLM Wiki |
|--------|-----|----------|
| **存储** | 向量数据库 | Markdown 文件 |
| **检索** | 相似度搜索 | 结构化查询 |
| **可读性** | ❌ 二进制向量 | ✅ 人类可读 |
| **可审计** | ❌ 困难 | ✅ 完全可查 |
| **复杂度** | 高 | 低 |
| **成本** | 高（向量库 + 嵌入） | 低（只需 LLM） |
| **可编辑** | ❌ 困难 | ✅ 直接编辑 |
| **可分享** | ❌ 困难 | ✅ Markdown 文件 |

## 💡 核心优势

### 1. 知识复利
```
每次学习都沉淀到知识库
    ↓
越用越有价值
    ↓
避免重复学习
```

### 2. 避免"失忆"
```
传统对话：每次重置上下文
LLM Wiki：持久化知识库
    ↓
不再"失忆"
```

### 3. 自动关联
```
AI 自动建立知识链接
    ↓
发现隐藏模式
    ↓
激发新洞察
```

### 4. 可传承
```
知识库可以分享
    ↓
团队协作基础
    ↓
长期积累价值
```
