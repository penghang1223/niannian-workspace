## ✅ 核心过稿要素

### 1. 开篇（前 3000 字）
| 要素 | 要求 | 权重 |
|------|------|------|
| **钩子** | 300 字内出现悬念/冲突 | ⭐⭐⭐⭐⭐ |
| **人设** | 主角性格清晰 | ⭐⭐⭐⭐ |
| **节奏** | 不拖沓，有推进 | ⭐⭐⭐⭐⭐ |
| **代入感** | 读者能共情 | ⭐⭐⭐⭐ |

### 2. 剧情（整体）
| 要素 | 要求 | 权重 |
|------|------|------|
| **爽点密度** | 每 2000 字至少 1 个爽点 | ⭐⭐⭐⭐⭐ |
| **反转** | 意料之外情理之中 | ⭐⭐⭐⭐ |
| **伏笔** | 有埋有收 | ⭐⭐⭐⭐ |
| **逻辑** | 无明显 bug | ⭐⭐⭐⭐⭐ |

### 3. 人物
| 要素 | 要求 | 权重 |
|------|------|------|
| **主角** | 动机清晰，成长弧完整 | ⭐⭐⭐⭐⭐ |
| **反派** | 不降智，有动机 | ⭐⭐⭐⭐ |
| **配角** | 有记忆点，不工具人 | ⭐⭐⭐ |

### 4. 文笔
| 要素 | 要求 | 权重 |
|------|------|------|
| **流畅度** | 阅读无障碍 | ⭐⭐⭐⭐⭐ |
| **对话** | 符合人设，不水 | ⭐⭐⭐⭐ |
| **描写** | 恰到好处，不啰嗦 | ⭐⭐⭐ |

---

## 📊 盐选评分维度

```
总分 100 分 = 
  开篇吸引力 (25 分) +
  剧情爽点 (30 分) +
  人物塑造 (20 分) +
  文笔流畅 (15 分) +
  创新程度 (10 分)
```

**过稿线**：≥70 分  
**推荐线**：≥85 分  
**爆款线**：≥95 分

---

## 🚫 常见拒稿原因

| 原因 | 占比 | 解决方案 |
|------|------|---------|
| 开篇太慢 | 35% | 300 字内出冲突 |
| 爽点不足 | 25% | 增加打脸/逆袭 |
| 人设模糊 | 15% | 明确主角动机 |
| 逻辑 bug | 10% | 大纲阶段检查 |
| 敏感内容 | 10% | 用敏感词清单自查 |
| 其他 | 5% | - |

## 📋 投稿前自检清单

- [ ] 开篇 300 字有钩子
- [ ] 每 2000 字有爽点
- [ ] 主角动机清晰
- [ ] 反派不降智
- [ ] 无明显逻辑 bug
- [ ] 无敏感词
- [ ] 字数达标（盐选一般 1-3 万字）
- [ ] 格式正确（分段/标点）
