## 💰 变现方式详解

### 方式 1：平台签约（最主流）

#### 知乎盐选

**特点**：
- 短篇为主（1-3 万字）
- 脑洞/反转/情感类
- 过稿后按阅读付费分成

**收益结构**：
```
收入 = 会员分成 + 单章购买 + 盐选专栏

会员分成：按阅读时长占比分配
单章购买：每章 0.1-0.5 元
盐选专栏：打包售卖，分成 50%
```

**过稿要求**：
- ✅ 开头 300 字必须有冲突
- ✅ 节奏快，每 2000 字一个爽点
- ✅ 反转至少 3 次
- ✅ 无敏感内容

**投稿方式**：
```
1. 知乎 APP → 创作中心 → 盐选创作者
2. 提交大纲 + 前 3 章
3. 编辑审核（7-15 天）
4. 签约 → 上架 → 收益
```

**预估收益**：
- 新手：1000-5000 元/月
- 中腰部：5000-20000 元/月
- 头部：20000-100000 元/月

---

#### 番茄小说

**特点**：
- 长篇为主（100 万字+）
- 免费流量，广告变现
- 按阅读时长分成

**收益结构**：
```
收入 = 广告分成 + 会员分成 + 打赏

广告分成：按阅读时长占比
会员分成：VIP 章节订阅
打赏：读者直接打赏
```

**签约要求**：
- ✅ 字数≥10 万字
- ✅ 日更≥4000 字
- ✅ 数据达标（阅读/留存）
- ✅ 无违规内容

**签约流程**：
```
1. 番茄作家助手 APP 注册
2. 发布作品（10 万字内）
3. 数据达标后编辑联系
4. 签约 → 持续更新 → 收益
```

**预估收益**：
- 新手：500-3000 元/月
- 中腰部：3000-15000 元/月
- 头部：15000-50000 元/月

---

#### 起点中文网

**特点**：
- 长篇为主（IP 向）
- 付费阅读模式
- 版权运营（影视/动漫/游戏）

**收益结构**：
```
收入 = 订阅分成 + 打赏 + 月票 + 版权

订阅分成：VIP 章节订阅，作者 50%
打赏：读者打赏，作者 50%
月票：月票榜奖励
版权：影视/动漫/游戏改编
```

**签约要求**：
- ✅ 字数≥3 万字
- ✅ 编辑内签或网站外签
- ✅ 更新稳定
- ✅ 无违规内容

**签约流程**：
```
1. 起点中文网注册
2. 发布作品（3 万字内）
3. 申请签约或编辑联系
4. 签约 → 上架 → 收益
```

**预估收益**：
- 新手：1000-5000 元/月
- 万订：10000-50000 元/月
- 大神：50000-500000 元/月 + 版权

### 方式 2：小说推广（门槛最低）

**特点**：
- 不用自己写小说
- 推广平台小说赚佣金
- 按引流付费阅读分成

**收益结构**：
```
收入 = 引流佣金 + 付费分成

引流佣金：每有效阅读 0.1-1 元
付费分成：用户付费阅读，分成 30-50%
```

**推广平台**：
| 平台 | 佣金 | 结算 | 门槛 |
|------|------|------|------|
| 知乎推文 | 0.5-1 元/人 | 日结 | 无 |
| 番茄推广 | 0.3-0.8 元/人 | 周结 | 无 |
| 七猫推广 | 0.2-0.6 元/人 | 周结 | 无 |
| 米读推广 | 0.2-0.5 元/人 | 周结 | 无 |

**推广方式**：
```
1. 抖音/快手：短视频引流
   - 小说精彩片段 + 配音
   - 引导点击链接阅读

2. 小红书：图文引流
   - 小说推荐笔记
   - 引导私信获取链接

3. 公众号：文章引流
   - 小说推荐文章
   - 阅读原文链接

4. B 站：视频引流
   - 小说解说视频
   - 简介区放链接
```

**预估收益**：
- 新手：100-1000 元/月
- 中腰部：1000-10000 元/月
- 头部：10000-50000 元/月

## 🚀 立即可用教程

### 知乎盐选过稿教程

**第 1 步：选题**
```
选择热门题材：
- 家庭伦理（全家反派/读心）
- 重生逆袭（重生后我 XXX）
- 系统爽文（系统绑定后）
- 悬疑反转（最后一刻反转）
```

**第 2 步：开篇**
```
开头 300 字必须有冲突：
❌ 错误：天气晴朗，我走在街上...
✅ 正确：婚礼当天，我发现老公出轨了...
```

**第 3 步：节奏**
```
每 2000 字一个爽点：
- 第 2000 字：第一次打脸
- 第 4000 字：第二次反转
- 第 6000 字：第三次高潮
```

**第 4 步：投稿**
```
1. 准备大纲（500 字）
2. 准备前 3 章（6000 字）
3. 知乎创作中心投稿
4. 等待审核（7-15 天）
```

## ⚠️ 避坑指南

### 常见骗局

| 骗局 | 识别方法 | 避免方式 |
|------|---------|---------|
| **收费签约** | 让你先交钱 | 正规平台不收费 |
| **高价收稿** | 承诺高价 | 市场价不会过高 |
| **盗版转载** | 未经同意转载 | 选择正规平台 |
| **AI 代写骗局** | 承诺包过 | 没有包过一说 |

---

### 注意事项

1. **不要一稿多投**
   - 签约后独家
   - 否则违约

2. **注意敏感内容**
   - 政治/色情/暴力
   - 平台会下架

3. **保持更新**
   - 断更影响收益
   - 读者会流失

4. **保护版权**
   - 及时签约
   - 保留创作记录
