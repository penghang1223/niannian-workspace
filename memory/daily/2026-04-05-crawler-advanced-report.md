## 🐛 避坑指南（已记录）

### 常见错误

| 错误 | 症状 | 解决方案 |
|------|------|----------|
| **只改 TLS 指纹** | 仍被 Cloudflare 拦截 | 必须组合浏览器 Stealth |
| **忽略行为暖身** | 初期成功，随后被封 | 添加 3-5 秒暖身行为 |
| **代理质量差** | 高延迟 + 高封禁率 | 使用移动端住宅代理 |
| **固定行为模式** | ML 模型识别为机器人 | 随机化所有行为参数 |
| **过度并发** | 短时间内大量请求 | 限制并发 + 添加延迟 |

### 最佳实践

1. **暖身时间**: 3-5 秒/页面
2. **请求间隔**: 2-5 秒随机
3. **并发限制**: 5-10 个并发/域名
4. **会话复用**: 同一会话 10-20 页后更换
5. **代理轮换**: 每 50-100 请求更换 IP
