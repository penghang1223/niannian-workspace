### 1. 编辑距离滚动数组优化

**标准版**:
```python
def levenshtein_distance(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]  # O(mn) 空间
    # ... 填充 DP 表
    return dp[m][n]
```

**优化版**:
```python
def levenshtein_distance_optimized(s1, s2):
    m, n = len(s1), len(s2)
    prev = list(range(n + 1))  # O(n) 空间
    curr = [0] * (n + 1)
    
    for i in range(1, m + 1):
        curr[0] = i
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                curr[j] = prev[j-1]
            else:
                curr[j] = 1 + min(prev[j], curr[j-1], prev[j-1])
        prev, curr = curr, prev
    
    return prev[n]  # 内存节省 99%+
```

### 2. 爬虫选型决策树（详细版）

```
开始
  ↓
目标网站类型？
  ↓
  ├── 静态网站（无 JS 执行）
  │     ↓
  │   有 TLS 指纹检测？
  │     ├── 是 → curl_cfinger
  │     └── 否 → requests + 代理
  │
  ├── 动态网站（需 JS 执行）
  │     ↓
  │   有 Cloudflare 保护？
  │     ├── 是 → Nodriver + 移动代理 + 行为暖身
  │     └── 否 → Playwright / Selenium
  │
  └── 大规模爬取（1000+ 页面）
        ↓
      有反爬措施？
        ├── 是 → Scrapy + curl_cfinger + 代理池
        └── 否 → Scrapy + 标准中间件
```

**选错代价案例**:
- 案例 1: 用 Playwright 爬取静态 API → 性能下降 10 倍（不必要 JS 执行）
- 案例 2: 用 requests 爬取动态网站 → 获取空内容（JS 未执行）
- 案例 3: 用 Scrapy 爬取 Cloudflare 保护网站 → IP 被封禁（无 TLS 指纹模拟）
