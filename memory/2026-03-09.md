## 💡 关键学习

### 飞书文档 API

**问题**：`feishu_doc` 的 `write` 和 `append` 动作不支持 docx 格式

**解决方案**：使用 `update_block` + `block_id` 来更新文档内容

**经验**：
- 飞书文档权限充足（1498 granted）
- 需要使用正确的 API 动作
- markdown 格式可以正常写入
