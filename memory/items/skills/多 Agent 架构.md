## 📚 参考框架

### ruvnet/ruflo (Claude-Flow)
- **Agent 数量**：64 个专用 Agent
- **分类**：16 个类别
- **特色**：蜂群智能、真理验证、训练管道

### OpenClaw 原生
- **创建方式**：sessions_spawn(runtime="subagent"|"acp")
- **管理方式**：subagents(action=list|steer|kill)
- **通信方式**：sessions_send

## 🎯 店小秘应用场景

### 1. 订单异常处理
```
协调器 → 判断 Agent → 分类处理 Agent → 结果整合
         ↓
    异常类型识别
    (物流/库存/支付)
```
**目标**：减少 80% 人工审核

### 2. 客服自动回复
```
协调器 → 理解 Agent → 知识库检索 → 回复生成 Agent
         ↓
    意图识别
    (咨询/投诉/建议)
```
**目标**：70% 自动回复率

### 3. 自动化测试
```
协调器 → 用例生成 Agent → 执行 Agent → 报告 Agent
         ↓
    基于需求生成测试用例
```
**目标**：覆盖率提升 50%

## 💡 实现要点

### Agent 设计原则
1. **单一职责** - 每个 Agent 专注一个任务
2. **可组合** - Agent 间可串联/并联
3. **可观测** - 执行过程可追踪
4. **可回滚** - 失败可恢复

### 协调器职责
1. 任务分解和依赖分析
2. Agent 选择和调度
3. 结果整合和质量检查
4. 异常处理和重试
