## 💡 爬虫选型决策树（玄机强调尤其实用）

根据玄机反馈，决策树价值突出：

```
目标网站类型？
    ├── 静态网站（无 JS）
    │   └── curl_cffi（TLS 指纹模拟）
    │
    ├── 动态网站（需 JS 执行）
    │   ├── 简单交互 → Playwright
    │   └── Cloudflare 保护 → Nodriver + 移动代理 + 行为暖身
    │
    └── 大规模爬取（1000+ 页面）
        └── Scrapy + curl_cfinger + 代理池
```

**选错代价案例**:
- 用 Playwright 爬静态 API → 性能下降 10 倍
- 用 requests 爬动态网站 → 获取空内容
- 用 Scrapy 爬 Cloudflare 保护 → IP 被封禁
