## 📝 使用说明

### 添加新记忆
```bash
# 偏好类
echo "内容" > items/preferences/新偏好.md

# 知识类  
echo "内容" > items/knowledge/新知识.md

# 重要对话
echo "内容" > resources/conversations/对话主题.md
```

### 日常日志
每天创建 `daily/YYYY-MM-DD.md`，记录当天的重要事件和对话。
