## 🔥 核心技术特点

### 1. 无需 JS 逆向
```
传统爬虫：逆向 JS 加密算法 → 技术门槛高
MediaCrawler: 利用登录态浏览器上下文 → 通过 JS 表达式获取签名
```

**优势**：
- ✅ 无需逆向复杂的加密算法
- ✅ 大幅降低技术门槛
- ✅ 基于 Playwright 浏览器自动化

### 2. 多平台支持
```
一套代码 → 支持 7 大主流平台
- 小红书、抖音、快手、B 站、微博、贴吧、知乎
```

### 3. 功能完整
```
- 关键词搜索爬取
- 指定帖子 ID 爬取
- 二级评论获取
- 创作者主页爬取
- 登录态缓存
- IP 代理池
- 评论词云图生成
```

### 场景 1：竞品分析 Agent 🔴 高价值

**需求**：
- 监控竞品公司的产品动态
- 分析竞品在社交媒体上的用户反馈
- 跟踪竞品营销活动和效果

**实施方案**：
```
MediaCrawler 爬虫
    ↓
小红书/抖音/微博数据
    ↓
AI 分析 Agent（情感分析 + 主题提取）
    ↓
竞品分析报告
    ↓
```

**技术栈**：
- MediaCrawler（数据采集）
- 年年的记忆系统（存储分析结果）
- LLM（情感分析、主题提取）
- 定期任务（心跳机制触发）
