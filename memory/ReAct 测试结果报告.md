## 🔧 优化建议

### 优化 1：改进复杂度分析

```python
def _analyze_complexity(self, query: str) -> float:
    score = 0.0
    
    # 因素 1：问题长度
    if len(query) > 30: score += 0.1
    if len(query) > 50: score += 0.2
    if len(query) > 100: score += 0.2
    
    # 因素 2：多子问题
    if query.count('?') + query.count('？') > 1: score += 0.2
    if any(word in query for word in ['并且', '还有', '另外', '同时']): score += 0.1
    
    # 因素 3：需要推理（扩展关键词）
    reasoning_words = [
        '为什么', '怎么', '如何', '怎样',  # 疑问词
        '分析', '评估', '对比', '比较', '异同',  # 分析类
        '方案', '策略', '计划', '建议',  # 规划类
        '因素', '考虑', '影响', '优缺点',  # 深度思考
    ]
    if any(word in query for word in reasoning_words):
        score += 0.3  # 提高到 0.3
    
    # 因素 4：涉及多领域
    domains = ['记忆', 'Agent', '架构', 'RAG', '检索', '评估', '协议', 'memU', 'ruflo']
    domain_count = sum(1 for d in domains if d in query)
    if domain_count > 1: score += 0.2
    
    return min(score, 1.0)
```
