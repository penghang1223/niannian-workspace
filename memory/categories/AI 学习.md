## 📚 参考项目

### ruvnet/ruflo (Claude-Flow)
- **定位**：企业级 AI 编排平台
- **核心特性**：
  - 64 个专用 Agent
  - 蜂群智能
  - 真理验证系统 (0.95 阈值)
  - 训练管道
- **借鉴点**：Agent 分类体系、验证机制

### NevaMind-AI/memU
- **定位**：24/7 主动式 Agent 记忆框架
- **核心理念**：文件系统式记忆组织
- **核心特性**：
  - 三层架构 (Resource→Item→Category)
  - 主动记忆提取
  - 双模式检索 (RAG + LLM)
- **借鉴点**：记忆结构、主动预测

### 其他关注项目
- X-PLUG/MobileAgent - GUI Agent
- microsoft/markitdown - 文档转 Markdown
- alibaba/OpenSandbox - AI 沙箱平台

## 💡 核心概念理解

### 多 Agent 架构
```
人类架构师 (主人)
    ↓
Agent 协调器
    ↓
┌─────────────────────────────┐
│ 需求 Agent → 开发 Agent     │
│ 测试 Agent → 部署 Agent     │
└─────────────────────────────┘
```

### 程序员角色转变
- **过去**：代码执行者
- **未来**：架构师 + 业务翻译者
- **核心能力**：需求理解、系统设计、质量把控
