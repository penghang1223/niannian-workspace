# 主动记忆提取规则

> 基于 memU 理念 | 年年的记忆提取指南 | 版本：1.0

---

## 🎯 提取目标

从与主人的对话中自动识别并提取：
- **偏好声明** - 主人的喜好、习惯、风格
- **意图目标** - 主人想要完成的任务/目标
- **知识点** - 新概念、新技能、新理解
- **关系信息** - 人、组织、服务配置
- **技能沉淀** - 可复用的方法、模式、经验

---

## 📋 提取规则

### 1️⃣ 偏好类 (preferences/)

**触发关键词**：
- "我喜欢..." / "我不喜欢..."
- "偏好..." / "习惯..."
- "用...比较好" / "...更方便"
- "希望..." / "想要..."

**示例**：
```
主人："我喜欢结构化的回答"
→ 提取：偏好/沟通风格 → 结构化呈现

主人："用 Markdown 比较好"
→ 提取：偏好/文档格式 → Markdown
```

**存储位置**：`items/preferences/`

---

### 2️⃣ 意图类 (intents/)

**触发关键词**：
- "帮我..." / "我要..."
- "需要..." / "想要..."
- "计划..." / "准备..."
- "目标是..." / "目的是..."

**示例**：
```
主人："帮我检索 GitHub 热门项目"
→ 提取：意图/短期任务 → GitHub 项目检索

主人："我想学习多 Agent 协同"
→ 提取：意图/学习目标 → 多 Agent 协同
```

**存储位置**：`items/intents/`

---

### 3️⃣ 知识类 (knowledge/)

**触发关键词**：
- "原来是..." / "明白了..."
- "概念是..." / "原理是..."
- "意思是..." / "理解为..."
- 学习/研究/理解 + 主题

**示例**：
```
主人学习 ruflo 项目后
→ 提取：知识/多 Agent 架构 → ruflo 核心特性

主人学习 memU 后
→ 提取：知识/记忆系统 → memU 文件系统理念
```

**存储位置**：`items/knowledge/`

---

### 4️⃣ 关系类 (relationships/)

**触发关键词**：
- 公司名、人名、组织名
- 配置、账号、API Key
- 服务、平台、工具名

**示例**：
```

主人："配置了 Brave API Key"
→ 提取：关系/服务配置 → Brave Search
```

**存储位置**：`items/relationships/`

---

### 5️⃣ 技能类 (skills/)

**触发关键词**：
- "学会了..." / "掌握了..."
- "可以..." / "能够..."
- "方法是..." / "步骤是..."
- 完成的任务、解决的问题

**示例**：
```
年年完成记忆系统重构
→ 提取：技能/年年能力 → 记忆系统重构

主人学习测试用例生成
→ 提取：技能/主人能力 → 测试用例设计
```

**存储位置**：`items/skills/`

## 🔗 交叉引用规则

### 自动建立链接的场景

1. **同一主题** - 新记忆与现有记忆主题相同
2. **引用提及** - 对话中提及已有记忆项
3. **上下位关系** - 新概念是已有概念的子类/父类
4. **相关场景** - 应用于相同场景或项目

### 链接格式
```markdown
[[../categories/AI 学习]]           # 引用分类
[[../items/preferences/沟通风格]]    # 引用记忆项
[[../../daily/2026-03-02]]          # 引用日志
```

## 🎯 提取流程

```
对话输入
    ↓
识别触发关键词
    ↓
分类判断 (偏好/意图/知识/关系/技能)
    ↓
提取核心内容
    ↓
检查现有记忆 (避免重复)
    ↓
创建/更新记忆项
    ↓
建立交叉引用
    ↓
更新 MEMORY.md 索引
```

---

## 💡 置信度评估

| 置信度 | 判断标准 | 处理方式 |
|--------|----------|----------|
| **高** | 主人明确表达、多次提及 | 直接创建记忆 |
| **中** | 隐含表达、上下文推断 | 创建但标注待确认 |
| **低** | 模糊表达、单次提及 | 暂存观察区 |

---

## 🚫 不提取的内容

- ❌ 临时性信息（"今天天气不错"）
- ❌ 未经确认的推测
- ❌ 敏感隐私信息（除非主人明确允许）
- ❌ 重复内容（已存在的记忆）
