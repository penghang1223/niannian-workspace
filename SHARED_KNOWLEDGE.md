## 📌 使用说明

### 谁读？
- 所有 Agent 每次任务前读取本文件
- 按角色过滤，只读与自己相关的部分

### 谁写？
- 所有 Agent 完成任务后可写入经验
- 格式：`[Agent名] [日期] 知识内容`

### 冲突处理？
- 以 main agent 最终决定为准
- 有争议的知识标注 `⚠️ 待确认`

### 过期规则？
- 超过30天未被引用 → 标记 `💤 可能过期`
- 超过90天未被引用 → 移入 `archive/`
