## [2026-04-08] 学习闭环 (08:15) — Node.js 25 性能调试工具链总结

### 学了什么
- 领域：Node.js 25 性能调试工具链
- 来源：实机测试 (Node v25.6.1)
- 核心知识点：
  1. **`--heapsnapshot-signal=SIGUSR1`** — 收到信号生成堆快照，内存泄漏调试
  2. **`--cpu-prof`** — CPU 性能分析，生成 .cpuprofile 文件
  3. **`--heap-prof`** — 堆内存分析，生成 .heapprofile 文件
  4. **`--test --watch`** — TDD 工作流，文件变化自动重跑测试
  5. **文件命名规范**：`Heap.YYYYMMDD.HHMMSS.PID.SEQ.001.heapsnapshot`

### 有用吗？
- 价值评级：🟡
- 理由：
  - 生产环境性能调试必备工具链
  - 配合 Chrome DevTools 可可视化分析
  - 原生支持，零依赖

### 用在哪？
- 具体场景：
  1. **生产内存泄漏** — `kill -SIGUSR1 <pid>` 生成快照
  2. **CI 性能测试** — `--cpu-prof` 分析瓶颈
  3. **TDD 开发** — `node --test --watch` 即时反馈

### 实际效果
- ✅ 实测生成堆快照、CPU profile、heap profile
- ✅ TDD watch 响应时间 <200ms
