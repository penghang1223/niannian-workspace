# 小说写作指令模板

## 世界观设定

### 基本信息
- **类型**: 东方玄幻 / 修真世界
- **时代背景**: 架空古代，灵气充沛的修真世界
- **核心冲突**: 正邪之争、天道与人道的抉择
- **力量体系**: 炼气→筑基→金丹→元婴→化神→渡劫→大乘

### 世界规则
- 灵气是修炼根本，天地间存在各种灵脉
- 修士可通过丹药、法宝、功法提升实力
- 天道有缺，每千年一次天劫考验
- 魔道以吞噬生灵精血修炼，为正派所不容

## 核心角色档案

### 主角：林墨
- **身份**: 青云宗外门弟子，实为千年难遇的"混沌灵根"
- **性格**: 沉稳内敛，重情重义，不轻易相信他人
- **当前境界**: 金丹后期（第42章时）
- **目标**: 查明师尊失踪真相，阻止魔道阴谋
- **语音特征**: 言简意赅，不善表达情感，但行动果断

### 女主：苏清雪
- **身份**: 天机阁圣女，精通推演之术
- **性格**: 清冷孤傲，内心善良
- **当前境界**: 元婴初期
- **与主角关系**: 亦敌亦友，逐渐产生情愫
- **语音特征**: 语气平淡，常用反问句

### 反派：血魔尊者
- **身份**: 魔道巨擘，谋划千年
- **目标**: 打开上古封印，释放域外天魔
- **当前境界**: 化神后期

## 故事梗概（前41章回顾）

林墨在青云宗外门苦修三年，意外觉醒混沌灵根，获得上古传承。师尊玄机子却在关键时刻失踪，留下神秘线索。林墨追查真相，发现魔道正在密谋打开上古封印。期间结识天机阁圣女苏清雪，两人从敌对到合作。第40-41章，林墨潜入魔道总坛，获得关键情报却被发现，重伤逃脱。

## 第42-46章大纲

### 第42章：绝境突围
- **关键事件**: 林墨带伤逃离魔道总坛，遭遇围追堵截
- **场景**: 血雾森林 → 地下暗河
- **目标**: 4000-5000字
- **重点**: 紧张刺激的追逃戏，体现主角机智和坚韧

### 第43章：疗伤与突破
- **关键事件**: 林墨在暗河中找到隐秘洞穴疗伤，意外突破至金丹巅峰
- **场景**: 地下洞穴 → 灵泉之眼
- **目标**: 4000-5000字
- **重点**: 描写突破过程，揭示混沌灵根的特殊之处

### 第44章：意外同盟
- **关键事件**: 苏清雪循着天机推演找到林墨，带来师尊下落的消息
- **场景**: 地下洞穴 → 返回地面
- **目标**: 4000-5000字
- **重点**: 两人关系深化，透露新的情报

### 第45章：师尊之谜
- **关键事件**: 苏清雪揭示玄机子被关押的地点，以及魔道的真正目的
- **场景**: 山洞密谈
- **目标**: 4000-5000字
- **重点**: 重大情报揭露，世界观扩展

### 第46章：制定反击
- **关键事件**: 林墨与苏清雪制定营救计划，决定联合正道各派
- **场景**: 山洞 → 启程
- **目标**: 4000-5000字
- **重点**: 承上启下，为后续大战铺垫

## 写作风格指南

### 叙事风格
- **视角**: 第三人称有限视角（跟随林墨）
- **时态**: 过去式
- **句式**: 长短结合，动作场景用短句增强节奏感
- **描写**: 偏重视觉和触觉，适当加入心理描写

### 对话风格
- 主角林墨：话少，直接，不绕弯子
- 苏清雪：清冷，说话带疏离感
- 反派：阴狠，喜欢嘲讽

### 禁忌
- 避免现代网络用语
- 避免突然出现的现代概念
- 保持修真世界的一致性

## 连续性要求

### 必须承接的前情
1. 林墨在第41章被血魔使击中左肩，有重伤
2. 他从魔道总坛偷出的情报玉简已损毁一半
3. 苏清雪与他约定三日后在天机峰见面

### 伏笔推进
1. 混沌灵根的秘密需要进一步揭示
2. 师尊玄机子的真实身份埋下更多线索
3. 血魔尊者的计划需要部分揭露

### 角色状态（第42章开始）
- 林墨：重伤、疲惫，但意志坚定
- 苏清雪：正在寻找林墨，状态完好
