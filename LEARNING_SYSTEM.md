## 🔄 学习执行机制

### Cron 定时任务
每个 Agent 的学习任务由 Cron 定时触发：
- 早晨学习：09:00 - 10:00
- 午后学习：14:00 - 15:00
- 晚间学习：20:00 - 21:00
- 深夜学习：23:00 - 00:00（轮换）

### 学习内容来源
1. **技能文档**：OpenClaw 官方文档 + ClawHub
2. **在线资源**：GitHub/技术博客/视频教程
3. **实践项目**：结合实际任务学习
4. **经验总结**：每日复盘 + Instinct 提取
