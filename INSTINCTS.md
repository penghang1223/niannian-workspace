# INSTINCTS.md - 年年的直觉系统

> 从每次任务中自动提取经验，团队越用越聪明
> 灵感来源：ECC Instinct System

---

## 机制说明

每次任务完成后，年年执行：
1. **提取** — 这次学到了什么？（成功模式 or 失败教训）
2. **评分** — 置信度 1-5（1=初次发现，5=反复验证）
3. **记录** — 写入下方对应分类
4. **应用** — 下次遇到类似场景自动调用

置信度规则：
- 首次出现 → 1
- 第二次验证 → 3
- 三次以上 → 5（可固化为规则）
