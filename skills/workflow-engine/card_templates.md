## 🎨 样式规范

### Emoji 使用规范

| 类型 | Emoji | 说明 |
|------|-------|------|
| 启动 | ✅ 🚀 | 开始/启动 |
| 进度 | ⏰ ⚠️ 🔴 | 提醒/催促/超时 |
| 决策 | 🤔 ✓ ✗ | 决策/通过/不通过 |
| 完成 | 🎉 🎊 ✅ | 完成/庆祝 |
| 状态 | 📊 📋 | 状态/列表 |
| 错误 | ❌ ⚠️ | 错误/警告 |

### 颜色规范

| 状态 | 颜色 | 说明 |
|------|------|------|
| 正常 | 🟢 绿色 | 进行中/完成 |
| 提醒 | 🟡 黄色 | 50% 进度提醒 |
| 催促 | 🟠 橙色 | 80% 进度催促 |
| 超时 | 🔴 红色 | 超时/错误 |
| 决策 | 🔵 蓝色 | 决策节点 |

## 📋 模板使用示例

### 示例 1：启动工作流

```python
send_card(
    template="workflow_start",
    novel_name="我在修仙界开网约车",
    project_id="novel_001",
    wave_name="Wave-1 选题定方向",
    status="started",
    first_step="望舒 - 分析题材数据",
    estimated_time="2 小时"
)
```

### 示例 2：进度催促

```python
send_card(
    template="progress_urge",
    task="创作第 1 章",
    elapsed=48,
    remaining=12
)
```

### 示例 3：波次完成

```python
send_card(
    template="wave_completion",
    workflow_name="选题定方向",
    wave_num=1,
    progress=25,
    elapsed_time="2 小时",
    next_step="Wave-2 大纲 + 人设"
)
```
