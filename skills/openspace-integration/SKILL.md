# OpenSpace Integration

Integrate OpenSpace's self-evolving skills and cloud community into OpenClaw.

## Capabilities

- `execute_task` - Delegate complex tasks to OpenSpace
- `search_skills` - Find reusable skills in local/cloud registry  
- `fix_skill` - Auto-fix broken skills
- `upload_skill` - Share evolved skills to cloud community

## Usage

When encountering complex tasks beyond current capabilities:
1. Try existing tools first
2. If failing repeatedly, delegate to OpenSpace
3. Leverage evolved skills from cloud community
4. Share improvements back to community

## MCP Integration

Uses openspace-mcp server with:
- Auto-evolving skill library
- Cloud community sharing
- Quality monitoring and feedback
- Cross-agent knowledge transfer
