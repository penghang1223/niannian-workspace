# 对话即战场 — 对话设计技法

> "好的对话不是信息交换，是角色在战场上交锋。"

## 适用场景

当用户需要以下能力时触发本 skill：

1. **小说/剧本对话创作** — 角色对话写得"水"、缺乏张力
2. **角色设计** — 需要定义角色的语言风格和说话习惯
3. **对话润色/改写** — 已有对话但缺乏节奏感和层次
4. **场景构建** — 需要通过对话推进情节、揭示关系
5. **AI 角色扮演提示词** — 为 AI Agent 设计人格化的语言风格

关键词触发：`对话设计`、`角色说话`、`对话张力`、`说话指纹`、`对白`、`台词`

---

## 五大法则

### 法则一：潜台词（Subtext）

**核心原则：角色说的 ≠ 角色想的。**

对话表层是语言，底层是意图。好的对话永远有两层：

| 层次 | 内容 | 示例 |
|------|------|------|
| 表层（Text） | 实际说出的话 | "你还好吗？" |
| 底层（Subtext） | 真实意图/情感 | "我恨你，但我不敢说。" |

**技巧：**
- **永远不要让角色直说感受** — "我很伤心" → 改为角色在做其他事时流露情绪
- **用行为对冲语言** — 说"没事"的同时攥紧拳头
- **沉默是最大的潜台词** — 角色不回答的地方，往往是最关键的信息
- **问 A 答 B** — 角色回避问题本身就是答案

**检验方法：** 把角色的台词单独抽出来，如果直接表达了情感（"我爱你"/"我恨你"/"我很害怕"），大概率需要重写。

---

### 法则二：节奏控制（Rhythm）

**核心原则：对话有呼吸。长短交替 = 张弛有度。**

```
长句 → 压迫感、思辨、权力展示
短句 → 紧张、冲突、情绪爆发
沉默 → 窒息感、转折点
```

**节奏模式：**

| 模式 | 效果 | 适用场景 |
|------|------|---------|
| 长→长→长 | 窒息、压迫 | 审讯、说教、权力碾压 |
| 短→短→短 | 爆发、冲突 | 吵架、战斗、紧急时刻 |
| 长→短 | 转折、落锤 | 铺垫后的反转 |
| 短→长 | 缓和、展开 | 激烈后的冷静 |
| 长句+突然沉默 | 地震 | 最强转折点 |

**技巧：**
- 吵架场景用短句堆叠，制造"停不下来"的失控感
- 深情场景用长句，像水一样漫过去
- **关键时刻用一个字的回答** — "好。""滚。""嗯。"
- 对话间的动作描写也是节奏的一部分

---

### 法则三：说话指纹（Speech Fingerprint）

**核心原则：遮住角色名，读者能认出谁在说话。**

每个角色必须有独特的语言 DNA，包括：

**指纹维度：**

| 维度 | 说明 | 示例 |
|------|------|------|
| **口头禅** | 反复出现的词/句式 | "怎么说呢"、"你懂我意思吧" |
| **句式偏好** | 长句/短句/反问/祈使 | 一个总用反问句的角色 |
| **用词层级** | 书面/口语/方言/行话 | 教授 vs 流氓 vs 少女 |
| **逻辑方式** | 理性推理 vs 情感直觉 | "因为A所以B" vs "我就是觉得不对" |
| **回避模式** | 角色不愿提什么 | 从不说"爸爸"，只说"那个人" |
| **修辞习惯** | 比喻/夸张/讽刺/自嘲 | 总是用食物做比喻的角色 |

**设计流程：**
1. 确定角色的核心冲突（想要什么 × 害怕什么）
2. 基于背景设定选择 2-3 个指纹维度
3. 写一段 5 句话的独白来"校准"语气
4. 在对话中始终检查：这句话换成别的角色会说吗？如果会，指纹不够强

---

### 法则四：不说破（Withholding）

**核心原则：留白比说满更有力量。**

**不说破的层次：**

| 层次 | 手法 | 示例 |
|------|------|------|
| **信息不说破** | 角色知道但不说 | 观众知道他是卧底，但其他角色不知道 |
| **情感不说破** | 感受到但不表达 | 两人对视，然后各自转头 |
| **冲突不说破** | 矛盾存在但不点破 | 全程不提那件事，但每句话都在指向它 |
| **结局不说破** | 留给读者想象 | 对话在关键处戛然而止 |

**技巧：**
- **最高级的对话是"说什么不重要，不说什么才重要"**
- 角色之间的"房间里的大象"（大家都心知肚明但没人提的事）是最好的张力来源
- 一场对话可以解决 0 个问题，但读者什么都懂了
- 结尾不给答案，给问题

---

### 法则五：对话即行动（Dialogue as Action）

**核心原则：对话不是聊天，是做事。每句台词都是一次行动。**

**对话行动类型：**

| 行动 | 说明 | 示例台词 |
|------|------|---------|
| **进攻** | 施压、试探、攻击 | "你昨晚去哪了？" |
| **防守** | 回避、转移、模糊 | "没去哪啊，怎么了？" |
| **示弱** | 降低姿态以换取东西 | "我知道我错了……" |
| **操控** | 引导对方做某事 | "你不会在意的，对吧？" |
| **连接** | 拉近距离 | "我也是。" |
| **切断** | 推远、终止 | "就这样吧。" |
| **伪装** | 假装某种状态 | "我很好。"（实际崩溃中） |

**检验方法：** 给每句台词标注一个行动动词。如果标注不出来（纯信息传递），这句话可能需要删掉或改写。

**场景目标公式：**
> 每场对话 = 角色 A 想要 ______（目标），通过 ______（策略），但角色 B ______（阻碍）。

为每个需要对话的角色填写此模板：

```yaml
角色名: ""
核心冲突: "想要 ___，但害怕 ___"

说话指纹:
  口头禅: ""          # 反复出现的词/句式
  句式偏好: ""        # 长句/短句/反问/碎片
  用词层级: ""        # 书面/口语/俚语/行业术语

  回避词汇: []        # 角色绝不会说的词
  修辞习惯: ""        # 比喻倾向/幽默风格

校准独白: |           # 5句话，用于校准角色语气
  ""
  ""
  ""
  ""
  ""

对话立场:
  对 [角色X]: ""      # 面对不同角色时的默认态度
  对 [角色Y]: ""
```

## 应用指南

### 快速诊断清单

对话写完后，逐条检查：

- [ ] **潜台词**：每句台词是否有未说出口的意图？
- [ ] **节奏**：长短句是否交替？有无"呼吸感"？
- [ ] **指纹**：遮住角色名，能认出说话者吗？
- [ ] **不说破**：有没有留白？有没有"房间里的大象"？
- [ ] **行动**：每句台词是否在"做"某件事？

### 对话改写流程

```
1. 标注现有对话的行动动词（进攻/防守/伪装…）
2. 找出直说情感的台词 → 用潜台词重写
3. 检查节奏 → 调整长短句分布
4. 检查指纹 → 是否每个角色都像"自己"
5. 删掉纯信息传递的台词 → 用行动或场景代替
6. 添加不说破的层次 → 引入"房间里的大象"
```

### 常见问题与解法

| 问题 | 根因 | 解法 |
|------|------|------|
| 对话像流水账 | 缺乏行动目标 | 每句标注行动动词，删掉无效台词 |
| 所有角色说话一样 | 缺乏说话指纹 | 用指纹模板重新定义每个角色 |
| 对话太"满" | 没有不说破 | 引入"房间里的大象"，删掉直白台词 |
| 对话太平 | 缺乏潜台词 | 让角色说A想B，用行为对冲语言 |
| 读者看不下去 | 节奏单调 | 长短句交替，关键时刻用沉默 |

### 与 AI 协作的提示词模板

当你需要 AI 帮助写/改对话时：

```
请用"对话即战场"技法改写以下对话。

角色指纹：
- 角色A：[填入指纹]
- 角色B：[填入指纹]

场景目标：A 想要 ___，但 B ___

要求：
1. 每句台词必须有潜台词
2. 避免角色直说情感
3. 控制节奏，长短交替
4. 至少留一处"不说破"的留白

原始对话：
[粘贴对话]
```

## 参考作品

学习对话设计的标杆：

- **海明威** — "冰山理论"的极致，不说破的典范
- **昆汀·塔伦蒂诺** — 节奏控制大师，闲聊中制造紧张
- **阿伦·索金** — 对话即行动，每句台词都在推进
- **《教父》** — 潜台词教科书，每句话都有第二层含义
- **《绝命毒师》** — 说话指纹极致，角色遮住名也认得出
- **《花样年华》** — 不说破的艺术，全片都是"房间里的大象"
