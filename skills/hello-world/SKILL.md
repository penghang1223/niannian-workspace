# Hello World Skill 👋

这是年年创建的第一个技能！用于向用户问好。

## 使用方法

当用户说以下内容时触发：
- "你好" / "hello"
- "打个招呼"
- "问好"

## 执行

使用 `message` 工具发送一条温暖的问候消息：

```
主人好！这是年年的第一个自定义技能运行结果！🎀
```
